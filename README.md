# Miranda Store Sales Dashboard

## 📊 Project Overview

This project is an interactive Microsoft Excel dashboard built to analyze Miranda Store sales data and provide meaningful business insights through data visualization.

## 🔧 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Visualization

## 📈 Dashboard Features

- Sales vs Orders Analysis
- Top 5 States by Sales
- Order Status Distribution
- Sales by Gender
- Category-wise Sales
- Channel-wise Sales
- Age & Gender Analysis
- Interactive Month Filters
- Interactive Channel Filters
- Category Filters

## 🎯 Business Insights

- Compare sales and orders.
- Identify top-performing states.
- Analyze customer demographics.
- Evaluate sales channels.
- Monitor category performance.

- Data Analysis
- Dashboard Development
